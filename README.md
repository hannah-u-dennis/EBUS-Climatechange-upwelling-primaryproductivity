# EBUS Upwelling and Productivity Analysis

This repository contains code used for the analysis in my MSc thesis:

"Changing Anthropogenic Climate Forcing and Primary Productivity in the Eastern Boundary Upwelling Systems"

## Data Sources
- OISSST
- ERA5 reanalysis wind stress
- CMEMS Chlorophyl-a
- CMIP6 experiments
  
## Methods
- Wind-based upwelling index
- SST-based upwelling index
- Ekman transport calculations
- Trend analysis
- Seasonal cycle analysis

## Study Regions
- California Current System
- Canary Current System
- Humboldt Current System
- Benguela Current System
